# 🛍️ Shopix

Shopix é uma loja virtual focada na venda de acessórios modernos como relógios, óculos, bolsas de ombro e joias.

O projeto foi desenvolvido utilizando tecnologias modernas para oferecer uma experiência rápida, responsiva e elegante.

---

# 🚀 Tecnologias

Este projeto foi construído com as seguintes tecnologias:

- ⚛️ React
- ⚡ Vite
- 🔷 TypeScript
- 🎨 TailwindCSS
- 🎬 Framer Motion
- 🧩 Lucide React
- 🗄️ Supabase (API + Banco de Dados)

Deploy:

- ▲ Vercel

---

# 📦 Funcionalidades

- Catálogo de produtos
- Interface moderna e responsiva
- Animações suaves
- Componentes reutilizáveis
- Integração com banco de dados
- Layout otimizado para mobile

# ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/diegorosa-x/shopix.git
```

Entre na pasta:

```bash
cd shopix
```

Instale as dependências:

```bash
npm install
```

---

# ▶️ Rodando o projeto

```bash
npm run dev
```

Aplicação disponível em:

```
http://localhost:5173
```

---

# 🏗️ Build para produção

```bash
npm run build
```

Preview do build:

```bash
npm run preview
```
